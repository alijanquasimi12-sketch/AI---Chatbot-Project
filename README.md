# ARIA - KMA² Intelligence Framework Chatbot

ARIA is an advanced, rule-based AI chatbot built on the proprietary KMA² Intelligence Framework. It is designed to provide highly contextual, multi-lingual, and mode-specific responses with an elegant, premium user interface.

## 🚀 Final Release Features (DecodeLabs Submission Version)

*   **Multi-Language Support**: Instantly toggle between English, Tamil, Telugu, Hindi, and Malayalam.
*   **Dynamic Mode Selector**: Customize the bot's behavior with 5 unique modes: Standard, Fast, Deep Thinking, Detailed Explanation, and Interview Mode.
*   **PDF & TXT Export Services**: Export entire conversation histories locally. PDF exports feature seamless auto-translation back to English for documentation purposes.
*   **Statistics Dashboard**: Real-time tracking of session length, total interactions, and user activity.
*   **Pinned Messages**: Pin important messages directly to the chat sidebar and persist them across sessions.
*   **ARIA Online Panel**: Real-time framework status, module readiness, and connection diagnostics.
*   **KMA² Developer Profile**: Premium glassmorphic UI overlay with a custom Silver Logo animation honoring the developer architecture.
*   **Robot Background System**: Floating, animated, semi-transparent robotic companions that react dynamically without overlapping chat bubbles.
*   **Settings Suite**: Comprehensive controls and quick-access panels to change configurations on the fly.
*   **100% Mobile Responsive**: Fluid UI architecture that scales beautifully from mobile devices to desktop monitors.

## 🛠️ Technology Stack
*   **Backend**: Python, Flask
*   **Frontend**: HTML5, Vanilla JavaScript, CSS3 (Glassmorphism, CSS Grid/Flexbox)
*   **Data Structure**: Proprietary KMA² Rule-Based Dictionary Engine

## 💻 Installation & Usage

1.  Clone this repository to your local machine.
2.  Install requirements:
    ```bash
    pip install -r requirements.txt
    ```
3.  Launch the application server:
    ```bash
    python app.py
    ```
4.  Open `http://localhost:5000` in any modern web browser to access ARIA.
